Windows desktop widget

See your Claude usage at a glance.

An always-on-top widget for your Claude subscription — 5-hour & weekly limits, live reset timers, per-model windows, and local token history with an API-cost estimate.

⬇ Download for Windows · .exe installer View source
Unsigned build — on first run Windows may show SmartScreen; click More info → Run anyway. Requires Windows 10/11 (WebView2).

What it shows

The same limits you see in Claude Code’s /usage, on your desktop.

⏱️

Live plan limits

5-hour and weekly utilization as ring gauges, with countdowns to each reset.

🧩

Per-model windows

Opus / Sonnet weekly limits shown separately when your plan has them.

📊

Token history + cost

Daily tokens from your local transcripts, with an estimated API-cost value.

🔀

Multiple sources

Switch between your Windows install and any WSL distro, or add a custom path.

🗕

Compact mode

Minimize to a tiny bar or ring view that stays out of your way.

📌

Always on top

Frameless, draggable from anywhere, and tucks into the system tray.

Your token stays on your machine

No accounts, no servers, no telemetry.

  • Reads your OAuth token from .credentials.json locally and uses it only to call api.anthropic.com/api/oauth/usage.
  • The token never leaves the Rust backend — the UI only receives utilization numbers and reset times.
  • The credentials file is read-only; token refresh is owned by Claude Code.
  • Fully open source — read every line before you run it.

Build it yourself

Prefer to compile from source?

  • Install Node, Rust (rustup, MSVC), and WebView2.
  • npm install
  • npm run tauri build
  • Installer lands in src-tauri/target/release/bundle/nsis/.
